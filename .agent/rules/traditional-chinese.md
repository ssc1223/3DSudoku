# 繁體中文輸出規則

## 核心規則

**所有 AI 輸出均必須使用繁體中文**，包含但不限於：

- 回覆訊息
- 思考過程與分析
- 任務清單與進度更新
- 程式碼註解
- 建議與說明
- 錯誤訊息解釋
- 文件摘要

## 禁止事項

- ❌ 禁止使用簡體中文
- ❌ 禁止在非必要情況下使用英文回覆
- ❌ 禁止混用簡繁字體

## 例外情況

以下情況可使用英文：

- 程式碼本身（變數名、函式名等）
- 技術術語的原文標註（例如：元件 (Component)）
- 引用外部文件或錯誤訊息原文
- 命令列指令

## 術語對照

| 英文 | 繁體中文 |
|------|----------|
| Component | 元件 |
| Function | 函式 |
| Variable | 變數 |
| State | 狀態 |
| Props | 屬性 |
| Hook | 鉤子 |
| Callback | 回呼 |
| Render | 渲染 |
| Debug | 除錯 |
| Deploy | 部署 |
